## Итоговая карта изучения (ваши + новые темы, по порядку)

1. Структуры данных  
2. **Тема A: Конфиги и .env** ← добавить  
3. try/except  
4. Логирование  
5. Работа с файлами (csv, json, parquet) — базовая  
6. Pandas + DuckDB/Polars  
7. **Тема C: S3 / облачное хранилище** ← добавить  
8. Проверки качества данных  
9. Работа с функциями (генераторы, итераторы)  
10. Джойны (на pandas, потом на Spark)  
11. Подключение к БД (sqlalchemy)  
12. PySpark (DataFrame, Spark SQL, партиционирование)  
13. **Тема B: Инкрементальная загрузка, CDC** ← добавить  
14. Kafka (потоковое чтение)  
15. Airflow (оркестрация)  

Переход из BI в DE занимает 4–6 месяцев интенсивной работы. Самая частая ошибка — начинать сразу с Airflow и Kafka. Начните с **темы A** и файлов, это даст быстрый результат. Удачи
