Вот полный список всех встроенных методов списков (list) в Python. Каждый метод изменяет исходный список (кроме операций получения длины или подсчета, которые просто возвращают значение).

### Добавление элементов

1.  **`.append(x)`** — добавляет один элемент `x` в конец списка.
    ```python
    a = [1, 2]
    a.append(3)      # [1, 2, 3]
    ```

2.  **`.extend(iterable)`** — расширяет список, добавляя все элементы из `iterable` (строки, списка, кортежа).
    ```python
    a = [1, 2]
    a.extend([3, 4]) # [1, 2, 3, 4]
    ```

3.  **`.insert(i, x)`** — вставляет элемент `x` на позицию `i`. Остальные элементы сдвигаются вправо.
    ```python
    a = [1, 3]
    a.insert(1, 2)   # [1, 2, 3]
    ```

### Удаление элементов

4.  **`.remove(x)`** — удаляет **первое** вхождение элемента со значением `x`. Вызывает `ValueError`, если элемент не найден.
    ```python
    a = [1, 2, 3, 2]
    a.remove(2)      # [1, 3, 2]
    ```

5.  **`.pop(i)`** — удаляет и **возвращает** элемент на позиции `i`. Если `i` не указана, удаляет последний элемент.
    ```python
    a = [1, 2, 3]
    x = a.pop(1)     # x = 2, a = [1, 3]
    last = a.pop()   # last = 3, a = [1]
    ```

6.  **`.clear()`** — удаляет **все** элементы из списка.
    ```python
    a = [1, 2, 3]
    a.clear()        # []
    ```

### Поиск и информация

7.  **`.index(x, start, end)`** — возвращает индекс **первого** вхождения `x`. Можно искать в срезе `[start:end]`. Если элемент не найден — `ValueError`.
    ```python
    a = [10, 20, 30, 20]
    i = a.index(20)  # i = 1
    ```

8.  **`.count(x)`** — возвращает количество вхождений элемента `x` в список.
    ```python
    a = [1, 2, 2, 3]
    c = a.count(2)   # c = 2
    ```

### Сортировка и порядок

9.  **`.sort(key=None, reverse=False)`** — сортирует список **на месте** (изменяет исходный).
    - `reverse=True` — по убыванию.
    - `key` — функция для ключа сравнения (например, `len`).
    ```python
    a = [3, 1, 2]
    a.sort()             # [1, 2, 3]
    a.sort(reverse=True) # [3, 2, 1]
    ```

10. **`.reverse()`** — переворачивает порядок элементов **на месте** (не сортирует, а просто разворачивает).
    ```python
    a = [1, 2, 3]
    a.reverse()          # [3, 2, 1]
    ```

### Копирование

11. **`.copy()`** — возвращает **поверхностную копию** списка.
    ```python
    a = [1, 2, 3]
    b = a.copy()         # b = [1, 2, 3] (новый объект)
    ```
    *Альтернатива: `b = a[:]` или `list(a)`.*

### Важное замечание

**Не путайте** методы списка со встроенными функциями, которые работают со списками, но возвращают новые объекты, не изменяя исходные:
- `sorted(list)` — возвращает новый отсортированный список (не меняет исходный).
- `reversed(list)` — возвращает итератор.
- `len(list)` — возвращает длину.
- `min(list)`, `max(list)`, `sum(list)`.

Пример различия:
```python
nums = [3, 1, 2]
nums.sort()      # изменил nums -> [1, 2, 3]

nums = [3, 1, 2]
new_nums = sorted(nums)  # nums остался [3, 1, 2], new_nums = [1, 2, 3]
```
