Вот полный список всех методов множеств (set) в Python:

## Базовые методы

| Метод | Описание | Пример |
|-------|----------|--------|
| `add(x)` | Добавляет элемент x в множество | `s.add(5)` |
| `remove(x)` | Удаляет элемент x (если нет - KeyError) | `s.remove(3)` |
| `discard(x)` | Удаляет элемент x (если нет - ничего не делает) | `s.discard(3)` |
| `pop()` | Удаляет и возвращает произвольный элемент | `s.pop()` |
| `clear()` | Удаляет все элементы из множества | `s.clear()` |
| `copy()` | Возвращает копию множества | `s2 = s.copy()` |

## Методы для операций над множествами

| Метод | Описание | Аналог оператора |
|-------|----------|------------------|
| `union(other)` | Объединение множеств | `\|` |
| `intersection(other)` | Пересечение множеств | `&` |
| `difference(other)` | Разность множеств | `-` |
| `symmetric_difference(other)` | Симметрическая разность | `^` |

## Методы для проверки отношений

| Метод | Описание |
|-------|----------|
| `issubset(other)` | Проверяет, является ли множество подмножеством | 
| `issuperset(other)` | Проверяет, является ли множество надмножеством |
| `isdisjoint(other)` | Проверяет, не имеют ли множества общих элементов |

## Методы для обновления множеств

| Метод | Описание | Аналог оператора |
|-------|----------|------------------|
| `update(other)` | Добавляет элементы из other | `\|=` |
| `intersection_update(other)` | Оставляет только общие элементы | `&=` |
| `difference_update(other)` | Удаляет элементы, присутствующие в other | `-=` |
| `symmetric_difference_update(other)` | Оставляет элементы из одного, но не обоих | `^=` |

## Примеры использования:

```python
# Создание множества
s = {1, 2, 3, 4}

# Добавление
s.add(5)              # {1, 2, 3, 4, 5}

# Удаление
s.remove(5)           # {1, 2, 3, 4}
s.discard(4)          # {1, 2, 3}
s.pop()               # удалит 1 (или любой другой)

# Операции с множествами
a = {1, 2, 3}
b = {3, 4, 5}

print(a.union(b))              # {1, 2, 3, 4, 5}
print(a.intersection(b))       # {3}
print(a.difference(b))         # {1, 2}
print(a.symmetric_difference(b)) # {1, 2, 4, 5}

# Проверки
print(a.issubset(b))           # False
print({1, 2}.issubset(a))      # True
print(a.isdisjoint({4, 5}))    # True

# Обновления
a.update(b)                    # a = {1, 2, 3, 4, 5}
a.intersection_update(b)       # a = {3, 4, 5}  
a.difference_update({4, 5})    # a = {3}
a.symmetric_difference_update({2, 3}) # a = {2}
```

## Важные замечания:
- Множества неупорядочены (элементы не имеют индексов)
- Элементы должны быть хэшируемыми (неизменяемыми)
- Множества не могут содержать дубликаты
- `frozenset` - неизменяемая версия множества (имеет меньше методов, только read-only операции)
